# Project overview

"Tfplanes" is a planes tracker app where you can add new flight or edit existing flight.

## Technologies
- Angular 11.2.8
- Typescript 4.1.5
- Rxjs 6.6.7
- Angular/Material 11.2.8
- Firebase 6.1.4
